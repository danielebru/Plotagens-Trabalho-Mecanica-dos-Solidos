# Plotagens-Trabalho-Mecanica-dos-Solidos
As plotagens apresentadas representam os gráficos das questões 1, 3 e 5 do Trabalho e mostram o comportamento da viga, com deslocamentos, deflexões e energia de deformação sob carga. Produzidas para a disciplina de Mecânica dos Sólidos, utilizam o Teorema de Castigliano e permitem comparar materiais e entender como cada parâmetro afeta a estrutura.

1. Requisitos

Deve-se ter instalado: 

-Python 3.8 ou superior

-bibliotecas:
numpy
pandas
matplotlib

Para instalar as dependências, use:

pip install numpy pandas matplotlib

2. Como salvar o código

1. Abra um editor de código(VS Code, PyCharm, Jupyter Notebook ou até o bloco de notas).

2. Copie o código apresentado no PDF.

3. Salve o arquivo com um nome, por exemplo:

questao_5_plot.py

4. Guarde o arquivo na pasta onde você deseja trabalhar.

 3. Como rodar o código

Abra o terminal/cmd na pasta onde o arquivo foi salvo e execute no terminal o nome do arquivo:

python questao_5_plot.py

Ou clique na seta, que indica "Run", na interface do VS Code, por exemplo.

Por fim, o programa realizará as plotagens e exibirá os cálculos no terminal.
